=== Communicator ===
Contributors: davidosomething
Requires at least: 3.2.1
Tested up to: 3.2.1

A simple chat plugin for WordPress dashboard.

== Description ==

Communicator allows members of your WordPress site to communicate 
with each other through a Dashboard widget. It is a super simple 
chat app for people who don't need everything BuddyPress has to offer.
